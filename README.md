# CookWise

A meal planning and grocery shopping optimization tool designed for the Swedish market.

## About

CookWise helps users plan their meals and optimize grocery shopping by suggesting recipes based on current sales and promotions at major Swedish grocery stores (ICA, Willys, Coop).

## Features

- Recipe suggestions based on items currently on sale
- Search and filter recipes by ingredients, cuisine, dietary restrictions, and budget
- Automatic shopping list generation from selected recipes
- Price comparison across different stores
- Map view with store locations and navigation
- User accounts with saved favorite recipes
